<h1 align="center">DIVA Sample IMIP Vue</h1>

该项目为智能制造园项目案例。

## 分支说明

- `main` 配合 DIVA 嵌入模式使用
- `streaming` 配合 DIVA 云渲染模式使用

## 下载

```shell
git clone https://github.com/sheencity/diva-sample-imip-vue2.git
```

## 安装依赖

- node v14 lts
- npm v7

```shell
npm install
```

## 运行项目

```shell
npm run serve
```

项目将运行在 `http://localhost:8080/`。

## 打包项目

```javascript
npm run build
```

打包文件将存储在 `dist/` 目录中。
